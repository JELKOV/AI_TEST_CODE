AI

AI Agent (AI 에이전트)

사람이 승인한 plan.md의 다음 행동 하나만 구현하고 실행 증거를 남긴다.

H

Human (사람)

계획과 기대 결과를 승인하고 테스트, diff, 설계를 최종 검토한다.

T

pytest (테스트 실행기)

합의된 행동을 실행 가능한 규칙으로 고정하고 회귀를 즉시 드러낸다.

TDD Cycle (테스트 주도 흐름)

RED → GREEN → REFACTOR

Canon TDD (원문) FastAPI Testing (공식 문서)
VIDEO 2:17 딩코딩코 Augmented Coding 해설·재현 계획 → RED → GREEN → REFACTOR → 검증
01

RED (실패 확인)

지급 비율 합계가 100이 아니면 422여야 한다는 테스트를 먼저 쓰고 실제 200 응답을 확인한다.

assert response.status_code == 422
02

GREEN (최소 구현)

선금·중도금·잔금의 합계가 정확히 100인지 확인하는 최소 규칙만 구현한다.

advance + interim + final == 100
03

REFACTOR (구조 개선)

GREEN에서 행동 변경과 분리해 이름과 구조만 개선하고 전체 테스트를 재실행한다.

uv run pytest -q
